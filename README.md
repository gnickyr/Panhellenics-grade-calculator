# Panhellenics-grade-calculator

This program is designed to calculate the score a student has earned by taking the Panhellenic Exams in Greece. 
In Greek high schools, students are initially divided into four tracks, each with a different set of courses.
For this reason, the program first asks the user to select their track.
Furthermore, Greek university departments assign different weighting factors to each subject.
For this reason, after the student selects a major, the program asks the user to enter the weighting factors for each course, depending on the school they wish to attend.
After that, the program asks for the exam scores and calculates the total points the student has earned.


Το πρόγραμμα αυτό είναι φτιαγμένο για να υπολογίζει τον βαθμό που έχει πάρει ένας μαθητής με την συμμετοχή του στις Πανελλαδικές στην Ελλάδα. 
Αρχικά στα Ελληνικά λύκεια, οι μαθητές χωρίζονται σε 4 κατευθύνσεις με διαφορετικά μαθήματα η κάθε μία.
Για αυτόν τον λόγο αρχικά το πρόγραμμα ζητάει από τον χρήστη να επιλέξει την κατεύθυνση του.
Ύστερα, οι Ελληνικές σχολές στα πανεπιστήμια έχουν διαφορετικούς συντελεστές βαρύτητας για το κάθε μάθημα.
Για αυτόν τον λόγο, μετά την επιλογή της κατεύθυνσης, το πρόγραμμα ζητάει απο τον χρήστη να εισάγει τους συντελεστές βαρύτητας του κάθε μαθήματος, ανάλογα με τη σχολή που θέλει.
Μετά από αυτό, το πρόγραμμα ζηταέι τους βαθμούς που έγραψε τις εξετάσεις, και υπολογίζει τα μόρια που έχει μαζέψει ο εξεταζόμενος.
